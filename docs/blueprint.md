# **App Name**: GiorBaliTour

## Core Features:

- Car Listing: Display available rental cars with details such as capacity, transmission, price, and image, ensuring only 'is_available' cars are shown.
- Service Reviews: Collect and display customer reviews for the overall GiorBaliTour service, showing userName, rating, and comment, enabling users to leave reviews with rating and comment inputs, with login required.
- Admin Dashboard: Provide a secure admin dashboard accessible at '/tanian' for CRUD operations on cars, including the ability to input image URLs and toggle car availability.
- Multi-language Support: Implement internationalization (i18n) with next-intl, supporting 8 languages (en, id, zh, ko, ar, tr, ru, pt) with 'en' as the default language.
- Authentication System: Set up an authentication system using Auth.js, including login and register pages, with user role checks performed server-side.

## Style Guidelines:

- Primary color: Dark Grey-900 to create a sophisticated and modern atmosphere.
- Secondary color: Yellow to simulate a glowing effect, adding depth and visual interest.
- Accent color: Electric blue (#7DF9FF) for interactive elements and highlights, ensuring they stand out.
- Headline font: 'Bebas Neue' (sans-serif) for titles, complemented with 'Roboto' (sans-serif) for body text; offers a modern and readable look.
- Use glowing, neon-style icons to match the overall theme, focusing on simplicity and recognizability.
- Implement a dark-themed layout with rounded elements and a focus on responsive design to ensure a seamless experience across devices.
- Add subtle transitions and loading states using glowing effects to enhance the user experience.
